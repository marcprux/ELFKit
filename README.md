## Features

- parse segments
- parse sections
- parse dynamics
- symbol list
- get all cstrings
- rebase infos
- notes
- ...

## Usage

### Load from file

For reading from file, use the `ELFFile` structure.

Reading from a file can be as follows.

```swift
let path = "Path to MachO file"
let url = URL(string: path)

let elf = try ELFFile(url: url)
```
